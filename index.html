1. 변수

1.1. 숫자형

코드 설명 기타
a, b = 1.5, 2 숫자형 변수 선언
a + b 숫자의 덧셈
a - b 숫자의 뺄셈
a * b 숫자의 곱셈
a / b 숫자의 나눗셈
a % b 나머지
a // b 몫
a ** b 거듭제곱

1.2. 문자형

코드 설명 기타
a = 'Hello!' 문자형 변수 선언
a + 'World\"' 문자열 합치기
a * 2 2번 반복
len(a) 문자열 길이 구하기
a[1:5], b[:3], c[2:-1] 문자열 인덱싱 및 슬라이싱
"I love %5s %-7d" % ('you', 3000) %를 이용한 문자열 포맷팅
"I love {0:*>10} {1}".format('you',3000) format을 이용한 포맷팅
f."I love {name}" f를 이용한 포맷팅 name = 'you'
a.find('l') 문자 l이 처음나온 위치 반환 ~존재 -> -1반환
a.index('l') 문자 l이 처음나온 위치 반환 ~존재 -> 에러

1.3. 리스트

코드 설명 기타
a = [1,2], b = list() 리스트형 변수 선언
a[1], b[:2] 문자열 인덱싱 및 슬라이싱
a + b 리스트 더하기
a * 3 리스트 반복
len(a) 리스트 길이 구하기
a[1] = 4 리스트 요소 수정
del a[0] 리스트 요소 삭제
a.append(1) 리스트 요소 추가
a.sort() 리스트 요소 순서대로 정렬
a.reverse() 리스트 역순
a.index(x) 리스트 내 x의 위치값 리턴 리스트 내 값 존재X -> 오류
a.insert(0,1) 0번 위치에 1삽입
a.remove(0) 처음 나오는 0삭제
a.pop(x) x번째 요소 반환후 삭제 매개인자 없으면 가장 마지막 값 반환,삭제
a.count(x) list내 x 값 개수 반환
a.extend(b) a와 b리스트 결합 매개인자는 오로지 리스트형만
